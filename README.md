# Decoupled-Learning-for-Conditional-Adversarial-Networks
Decoupled Learning for Conditional Adversarial Networks

## Pre-requisites
* Python 2.7x
* TensorFlow 1.x
* Matlab
* pynvml

## Adaptation on [Pix2Pix](https://github.com/phillipi/pix2pix)
Training
```
$ cd Adapt_Pix2Pix
$ python main.py
```

## Adaptation on [CAAE](https://github.com/ZZUTK/Face-Aging-CAAE)
Training
```
$ cd Adapt_CAAE
$ python main.py
```
